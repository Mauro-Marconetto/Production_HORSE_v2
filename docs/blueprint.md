# **App Name**: ForgeFlow

## Core Features:

- Automated Production Planning: Generates an optimized production plan, suggesting which mold goes on each machine per week to meet demand, respect stock levels, capacity, and downtime, using AI as a tool.
- Manual Plan Override: Allows manual adjustments to the suggested plan, providing flexibility and control.
- KPI Dashboard: Displays key performance indicators such as coverage per piece, machine utilization, mold changes, and potential stockout risks.
- Demand Forecasting: Imports demand data from CSV/Excel files, allowing versioning and freezing of monthly/weekly demand forecasts.
- Inventory Management: Defines stock levels, registers current stock, and provides alerts for critical stock percentages.
- Downtime Analysis: Tracks machine downtime, categorizes causes, and calculates effective machine availability based on historical averages.
- What-If Scenario Planning: Allows simulation of different production scenarios by adjusting OEE, scrap rates, and shifts to evaluate potential outcomes.

## Style Guidelines:

- Primary color: Steel Blue (#4682B4), representing the strength and reliability of the manufacturing process.
- Background color: Light Gray (#F0F8FF), provides a clean, neutral backdrop for data visualization.
- Accent color: Goldenrod (#DAA520), highlighting key metrics and actionable items for a premium feel.
- Headline font: 'Space Grotesk', sans-serif, for a techy and scientific feel; body font: 'Inter', sans-serif, for longer text passages.
- Use clear and concise icons to represent different modules and actions within the application.
- Employs a modular grid system to maintain consistency and responsiveness across different screen sizes.
- Subtle transitions and animations to provide feedback and enhance user experience, such as loading spinners or confirmation messages.
